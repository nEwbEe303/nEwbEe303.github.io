题目

Challenge 6 Set 1 - The Cryptopals Crypto Challenges
原题链接：http://www.cryptopals.com/sets/1/challenges/6

攻击重复密钥 XOR 加密

解密一个经过重复密钥 XOR 加密后又经过 base64 编码的文件。点此查看文件（文件内容为 base64 编码的密文）

中文版

给定一个经过重复密钥 XOR 加密后再进行 Base64 编码的文件，需要破解加密并还原出原始明文。重复密钥 XOR 是指使用一个循环重复的密钥对明文进行 XOR 加密。破解的关键在于：

  • 首先猜测密钥长度
  • 然后按位置拆分密文，每个位置使用单字节 XOR 破解
  • 最后组合成完整密钥并解密

解析

题目给出了完整的解题流程，我们的着手点是密钥长度。思路如下：

  1. 暴力破解密钥长度（2-40），使用汉明距离评估，归一化后最小的即为最可能的密钥长度；
  2. 根据密钥长度将密文划分为若干块，由于不同块的同一位置的字节与密钥中相应位置的字节进行单字节 XOR 加密，因此可以将不同块的同一位置的字节组成一个序列，采用 Challenge 3 中的单字节 XOR 解密方法，解密出密钥中的单个字节；
  3. 将所有位置的密钥字节连接起来，得到完整密钥；
  4. 使用得到的密钥对密文进行重复密钥 XOR 解密，得到明文。

知识点

汉明距离（Hamming Distance）

在信息论中，两个等长字符串之间的汉明距离是两个字符串对应位置的不同字符的个数。换句话说，它就是将一个字符串变换成另外一个字符串所需要替换的字符个数。在计算机中，则是一个 01 序列变成另一个 01 序列所需要替换的 01 个数。

性质：如果 a = b ⊕ c，那么 b = a ⊕ c。XOR 运算具有可逆性。

汉明距离的计算方法：将两个字节进行 XOR，然后统计结果中二进制位为 1 的个数。

为什么使用汉明距离猜测密钥长度？

当我们把密文按照猜测的密钥长度去划分成等长的块时，由于明文都是字母、空格、标点等 ASCII 字符，不同块之间的汉明距离会较小。因此，我们可以遍历可能的密钥长度，计算归一化后的平均汉明距离，最小值对应的长度最有可能是真实的密钥长度。

分析过程

  • 步骤1：读取文件并解码 - 文件内容为 Base64 编码，需要先解码为字节数据。
  • 步骤2：猜测密钥长度 - 遍历 2 到 40 之间的每个密钥长度，将密文按该长度分块，取前 4-6 块计算两两之间的汉明距离，归一化后取平均值，选择平均值最小的长度作为密钥长度。
  • 步骤3：分解密文 - 按照猜测的密钥长度将密文分成若干块，每块长度等于密钥长度。
  • 步骤4：破解每个密钥字节 - 对于密钥中的每个位置，从所有块中取出该位置的字节组成一个新序列，这个序列实际上是用单个密钥字节 XOR 加密的密文。使用 Challenge 3 中的方法（英文频率分析）破解出该密钥字节。
  • 步骤5：解密 - 使用得到的完整密钥对原始密文进行重复密钥 XOR 解密，得到明文。
💡 注意：在计算汉明距离时，需要将结果除以密钥长度进行归一化，这样才能在不同长度下进行比较。同时，取多组块的平均值可以使结果更加准确。

解决方案

使用 Python 实现上述流程。核心函数包括：

  • hamming_distance(x, y) - 计算两个字节串的汉明距离
  • get_key_size(ciphertext) - 猜测密钥长度
  • break_single_byte_xor(ciphertext) - 破解单字节 XOR（复用 Challenge 3）
  • repeating_xor(plaintext, key) - 重复密钥 XOR 加/解密

完整代码

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
# Cryptopals Challenge 6 - Break repeating-key XOR

from base64 import b64decode
import itertools

# ============================================================
# 辅助函数：汉明距离
# ============================================================
def hamming_distance(x: bytes, y: bytes) -> int:
    """计算两个等长字节串的汉明距离（不同bit的个数）"""
    assert len(x) == len(y)
    distance = 0
    for b1, b2 in zip(x, y):
        xor_result = b1 ^ b2
        distance += bin(xor_result).count('1')
    return distance


# ============================================================
# 步骤1：猜测密钥长度
# ============================================================
def guess_key_size(ciphertext: bytes, min_size=2, max_size=41, num_blocks=4):
    """
    通过汉明距离猜测密钥长度
    返回最可能的密钥长度列表（按分数排序）
    """
    key_scores = []
    
    for key_len in range(min_size, max_size):
        # 取前 num_blocks*2 个块，计算平均汉明距离
        blocks = []
        for i in range(num_blocks * 2):
            start = i * key_len
            end = start + key_len
            if end <= len(ciphertext):
                blocks.append(ciphertext[start:end])
        
        if len(blocks) < 2:
            continue
        
        # 计算所有块对之间的平均汉明距离
        total_distance = 0
        pairs = 0
        for i in range(len(blocks)):
            for j in range(i + 1, len(blocks)):
                total_distance += hamming_distance(blocks[i], blocks[j])
                pairs += 1
        
        if pairs == 0:
            continue
        
        avg_distance = total_distance / pairs
        # 归一化：除以密钥长度，得到每个字节的平均汉明距离
        normalized = avg_distance / key_len
        key_scores.append((key_len, normalized))
    
    # 按归一化汉明距离从小到大排序
    key_scores.sort(key=lambda x: x[1])
    return [ks[0] for ks in key_scores[:5]]  # 返回前5个最可能的长度


# ============================================================
# 步骤2：破解单字节 XOR（复用 Challenge 3 的方法）
# ============================================================
def score_english(text: bytes) -> float:
    """
    对字节串进行英文评分
    常见英文字母频率（基于大量英文文本）
    """
    # 英文字母频率表（百分比）
    freq = {
        b'e': 12.70, b't': 9.06, b'a': 8.17, b'o': 7.51, b'i': 6.97,
        b'n': 6.75, b's': 6.33, b'h': 6.09, b'r': 5.99, b'd': 4.25,
        b'l': 4.03, b'c': 2.78, b'u': 2.76, b'm': 2.41, b'w': 2.36,
        b'f': 2.23, b'g': 2.02, b'y': 1.97, b'p': 1.93, b'b': 1.49,
        b'v': 0.98, b'k': 0.77, b'j': 0.15, b'x': 0.15, b'q': 0.10, b'z': 0.07,
        b' ': 18.00  # 空格频率最高
    }
    
    score = 0
    for byte in text:
        ch = bytes([byte])
        if ch in freq:
            score += freq[ch]
        elif 65 <= byte <= 90 or 97 <= byte <= 122:
            # 字母但不在频率表中（大写转小写处理）
            lower = bytes([byte + 32 if 65 <= byte <= 90 else byte])
            if lower in freq:
                score += freq[lower] * 0.8  # 大写字母略降权
            else:
                score += 0.5
        elif byte == 10 or byte == 13:  # 换行符/回车
            score += 0.5
        elif 32 <= byte <= 126:  # 可打印字符
            score += 0.2
        else:
            score -= 10  # 非可打印字符惩罚
    return score


def break_single_byte_xor(ciphertext: bytes):
    """
    破解单字节 XOR 加密
    返回 (密钥字节, 解密后的明文, 得分)
    """
    best_score = -1
    best_key = 0
    best_plaintext = b''
    
    for key in range(256):
        decrypted = bytes([b ^ key for b in ciphertext])
        s = score_english(decrypted)
        if s > best_score:
            best_score = s
            best_key = key
            best_plaintext = decrypted
    
    return best_key, best_plaintext, best_score


# ============================================================
# 步骤3：重复密钥 XOR 加/解密
# ============================================================
def repeating_xor(data: bytes, key: bytes) -> bytes:
    """重复密钥 XOR 加/解密"""
    result = bytearray()
    for i, byte in enumerate(data):
        result.append(byte ^ key[i % len(key)])
    return bytes(result)
