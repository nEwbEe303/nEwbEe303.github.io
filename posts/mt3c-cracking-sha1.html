题目

cracking-sha1-hashed-passwords
原题链接：https://www.mysterytwisterc3.org/en/challenges/level-2/cracking-sha1-hashed-passwords

已知 password 经过 sha1-hash 后的值和键盘输入痕迹，破解密码。

解析

知识点

  • SHA-1（Secure Hash Algorithm 1）是一种密码散列函数，美国国家安全局设计，并由美国国家标准技术研究所（NIST）发布为联邦资料处理标准（FIPS）。SHA-1可以生成一个被称为消息摘要的160位（20字节）散列值，散列值通常的呈现形式为40个十六进制数。

分析

根据键盘上的指纹痕迹，我们可以判断密码所包含的字符：

['q', 'Q', 'w', 'W', 'n', 'N', 'i', 'I', '+', '*', '5', '%', '8', '(', ')', '0', '=', '1']

以及最小的密码长度为 8。注意到小键盘上只有 2、4、6、8 有痕迹，很有可能代表上下左右。穷举即可，从所有字符串中选出密码长度个（可重复）字符，组成密码，进行 SHA-1 运算，结果与给定的密码 SHA-1 值相比较，相同即可得到答案。

itertools.product 详解

product 是 Python 标准库中 itertools 模块中的一个函数。它用于计算多个可选对象的笛卡尔积。简而言之，product 返回所有可能的组合，其中每个组合由输入的可选对象中的一个元素组成。

itertools.product(*iterables, repeat=1)
  • *iterables：这是一个可变参数，你可以传递一个或多个可选对象作为参数。这些可选对象的元素将被用来生成组合。
  • repeat：这是一个可选参数，默认值为 1。它表示每个输入可选对象要重复使用的次数。如果设置为大于 1 的值，生成的组合中的每个元素都会在输出中重复多次。

示例：

import itertools

# 基本用法：两个列表的笛卡尔积
result = list(itertools.product(['a', 'b'], ['1', '2']))
# 结果: [('a','1'), ('a','2'), ('b','1'), ('b','2')]

# 使用 repeat 参数生成重复组合
result = list(itertools.product(['0', '1'], repeat=3))
# 结果: 000,001,010,011,100,101,110,111 (8种组合)

解决方案

使用 Python 的 itertools.product 生成所有可能的候选密码组合（字符集长度 8～20），计算 SHA-1 哈希并与目标值比对。

代码

# -*- coding: utf-8 -*-
import hashlib
import itertools

def is_equal(c, target_hash):
    return hashlib.sha1(c.encode('utf8')).hexdigest() == target_hash

def main():
    # 候选字符集（根据键盘指纹痕迹）
    charset = ['q', 'Q', 'w', 'W', 'n', 'N', 'i', 'I', '+', '*', '5', '%', '8', '(', ')', '0', '=', '1']
    target = '替换为题目给定的sha1哈希值'   # 实际挑战中提供的哈希
    
    for length in range(8, 20):
        for combo in itertools.product(charset, repeat=length):
            candidate = ''.join(combo)
            if is_equal(candidate, target):
                print("The password is :", candidate)
                return
    print("未找到匹配密码")

if __name__ == '__main__':
    main()

运行结果

The password is : (Q=win*5)
